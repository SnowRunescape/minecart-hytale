# Minecart Hytale

## What is it?

**Minecart Hytale** is a plugin for Hytale servers that integrates the **Minecart** virtual store directly into your server.

## What is it for?

This plugin performs **automatic delivery of purchases** made on the Minecart website to your Hytale server. When a player buys something from the virtual store, the plugin ensures they receive their items, passes, and benefits in-game quickly and securely.
